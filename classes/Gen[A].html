  • [weight=0.5]
)
GenA chainable

Defined in jsqc.js:582

Combine two generators into one, prefering one depending on weight. By default, there is a 50/50 split.

Parameters:

  • gen GenA

    Another generator from which to choose values.

  • [weight=0.5] Float optional

    How much this generator should be preferred over the passed in one.

Returns:

GenA: A generator that produces a mixture of values.

Example:

Gen.genUint.mixWith(Gen.genNull, 0.95); // Generate null 5% of the time.

genFloat

GenFloat static

Defined in jsqc.js:1120

Size scaled floats. Typically Gen/chooseFloat will be preferable. Only produces "well-behaved" floats. See Gen/genFloatSpecial or Gen/genFloatMix for more hostile tests.

Use Gen/suchThat to exclude 0 and such.
